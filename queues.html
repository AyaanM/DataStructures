A queue is a data structure which contains an ordered set of data

Queues provide 3 methods of interaction:

  • Enqueue - adds data to the “back” or end of the queue
  • Dequeue - provides and removes data from the “front” or beginning of the queue
  • Peek - reveals data from the “front” of the queue without removing it

A queue also can also be seen as a line to get tickets. In the Enque image, a person is being enqued in the line. In peek a person is being peeked at. In pop, a person is being dequed off the line


Just like stacks, queues can be implemented using a linked list as the underlying data structure because it’s more efficient than a list or array.
