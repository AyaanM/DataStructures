Singular Node

On the page we will be descussing how Nodes work

Nodes are the fundamental building blocks of many computer science data structures. They form the basis for linked lists, stacks, queues, trees, and more.

An individual Node contains data and a link to another node

The data inside of Nodes can either be a string, an integer, a float, an array, or null

Use this image of a singular node as an example. node_a contains the data as the integer 5, and it also contains a link to node_b


The links inside of a node can be called a pointer since it points to a node. When there are many nodes, a LinkedList is formed.

A LinkedList can contain of many nodes linked to each other. If there is no link null in a node then we have reached the end of our Linked List or the particular path we were following

Take the picture of linked list as example. Here there are many nodes and they link to many different nodes. If we were to track the path of node_a that would lead us to node_b but because node_b does not link to a node the path ends
